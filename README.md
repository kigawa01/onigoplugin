# onigoplugin

マイクラで鬼ごっこをするためのプラグイン

## Requirement

* Spigot-1.16.5

## Command

- **/onigo**
    - ゲームに関するコマンド
- **/onigocreate**
    - ゲームの作成に関するコマンド
- **/stage**
    - ステージに関するコマンド
